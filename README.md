# Set Up

### Clone the repo.

```
git clone https://github.com/${GitHub Username}/polygon-roadshow.git
```

<br>

### Open Terminal on the Folder

```
cd polygon-roadshow
```

<br>

### Install the Dependencies

```
npm install
```

<br>

### Run the UI

```
npm run dev
or
yarn start
```
